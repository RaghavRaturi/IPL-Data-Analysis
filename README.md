# IPL Data Analysis Dashboard (2008–2025)

📊 Project Overview

The IPL Data Analysis Dashboard (2008–2025) is an interactive Power BI dashboard that provides comprehensive insights into the Indian Premier League across multiple seasons. The dashboard enables users to explore season-wise statistics, team performances, player achievements, and tournament trends through dynamic visualizations.

The project focuses on transforming raw IPL data into meaningful business intelligence, helping users analyze historical performance and uncover key cricketing insights.

🚀 Features
Season Analysis
- Select any IPL season from 2008 to 2025
- View season-specific statistics and performance metrics
- Dynamic filtering and interactive visualizations

Tournament Summary
- Champion and Runner-up teams
- Total matches played
- Number of participating teams
- Total venues used

Batting Insights
- Orange Cap winner (Highest Run Scorer)
- Most runs scored in a season
- Top boundary hitters:
  * Most 4s
  * Most 6s
- Total season boundaries:
  *Total Fours
  *Total Sixes

Bowling Insights
- Purple Cap winner (Highest Wicket Taker)
- Most wickets in a season

Performance Metrics
- Total centuries
- Total half-centuries
- Team-wise points table
- Wins, losses, no results, and total points
